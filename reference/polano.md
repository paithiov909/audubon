# Whole text of 'Porano no Hiroba' written by Miyazawa Kenji from Aozora Bunko

Whole text of 'Porano no Hiroba' written by Miyazawa Kenji from Aozora
Bunko

## Usage

``` r
polano
```

## Format

An object of class `character` of length 899.

## Source

<https://www.aozora.gr.jp/cards/000081/files/1935_ruby_19924.zip>

## Details

A dataset containing the text of Miyazawa Kenji's novel "Porano no
Hiroba" which was published in 1934, the year after Kenji's death.
Copyright of this work has expired since more than 70 years have passed
after the author's death.

The UTF-8 plain text is sourced from
<https://www.aozora.gr.jp/cards/000081/card1935.html> and is cleaned of
meta data.
